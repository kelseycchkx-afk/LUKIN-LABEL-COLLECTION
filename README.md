# LUKIN-LABEL-COLLECTION
- Collect Luckin label from NO. 001 to NO. 999
- A mobile-friendly web app for creating and collecting Luckin coffee label stickers.

## Features

- Create a digital Luckin label from manually entered information.
- Use photo recognition to help fill in label fields automatically.
- Preview the generated label in real time.
- Export the label as a PNG image.
- Save collected labels locally in the browser.
- View collected labels sorted by label number.
- Browse a 001-999 collection grid.
- Tap a collected label in the grid to preview it.
- If a label number already exists, the app asks whether to overwrite the old record.

## GitHub Pages

This app is designed to run as a static GitHub Pages site.

After deployment, open the GitHub Pages URL on your phone or computer. The QR code generated on each label will automatically point to the current app URL.

## Data Storage

All saved labels are stored locally in the browser. No label data is uploaded to a server.

## Notes

Photo recognition uses browser-based OCR. The first recognition may require an internet connection to load OCR resources.
